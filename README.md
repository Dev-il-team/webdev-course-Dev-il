# Fundamentos de Desarrollo Web

## Resumen del Curso

Este curso de aproximadamente una hora introduce a estudiantes de secundaria al desarrollo web utilizando HTML y CSS.

**¡No requiere descargas!** Solo necesitas un navegador web y herramientas gratuitas en línea.

**Duración total:** ~60 minutos

**Público objetivo:** Estudiantes de 12 a 17 años sin experiencia en programación.

**Prerrequisitos:** Ninguno.

**Herramientas necesarias:** Navegador web (Chrome, Firefox, Edge o Safari), CodePen, JSFiddle y Replit.

**📂 Repositorio de código fuente:** https://github.com/Dev-il-team/webdev-course-Dev-il
